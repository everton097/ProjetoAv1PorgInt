# Avaliação 01 de Programação para Internet I

Este repositório serve para a avaliação 01 de programação para internet I.

```
Etapa 1 da avaliação.

Escolha um site de sua preferência (e.g. Instagram, Youtube) e tente criar uma cópia dele utilizando apenas HTML. Seu site deve:

 1. conter no mínimo 5 páginas e 2 diretórios;
 2. seguir o guia de estilo de HTML descrito no W3Schools;
 3. deve ser possível navegar entre as páginas através de links;
 4. todas as páginas devem conter títulos (headings);
 5. todas as páginas devem conter parágrafos formatados;
 6. todas as páginas devem conter imagens;
 7. pelo menos uma página deve conter uma tabela;
 8. pelo menos uma página deve conter um formulário;
 9. pelo menos uma página deve conter uma lista ordenada;
 10. pelo menos uma página deve conter uma lista desordenada .

Cada critério vale 10% da nota da tarefa.
```
